How to Create Github Pages By Following Git System?

3 November, 2021
0 Comments

A. Static Website Creation Steps

1. The first step, Search for HTML & CSS templates at https://www.free-css.com/free-css-templates. On the website, there are many recommendations for HTML & CSS website templates and we can use the website for free.

2. The second step, After finding the desired website template, then download it. Usually the file downloaded from the web is still in the form of WInRar so we need to extract the file first.

3. The third step, then we can edit a web page using notepad++, vscode, sublime, atom, metbeans, etc.

B. Steps to create a githubpage using the git system.

1. Download and Install Git on the following website link:(https://git-scm.com/download/win).

2. Create a new repository by clicking on new repository as shown in the image below. Then fill in the repository name with username.github.io, then for the description name, it's up to you to write the task description or leave it blank. Then click Create respository to create the repository you want to create.

3. Then go back to the web folder that has been created to add to github, right click and select “Git Bash Here”. Then initiate the respository using “git init”

4. After that do the “git clone” command to clone from the github repository to local. Then copy the entire web file that is used and put it in the nnoroll.github.io file

5. Do the command "git add" to add all files and directories to be committed.

6. Lalu lakukan perintah “git commit –m ‘first commit’ ” karena ini merupakan commit pertama jadinya memberikan pesan berupa first commit

7. The last step is to do "git push -u origin master" to push local files into the github repository.

8. Then we check on github and refresh it, if it looks like below then it can be said that the local git repository has been successfully cloned to the Github repository.

9. To be sure of the latter, check the website with the address nnoroll.github.io . We can see in the picture on the side that the website is already accessible.
